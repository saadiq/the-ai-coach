# The AI Coach

AI transformation consulting landing page built with Astro and deployed to Cloudflare Pages.

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun dev

# Build for production
bun run build

# Preview production build
bun run preview
```

Open [http://localhost:4321](http://localhost:4321) to view the site.

## Deployment

This site is deployed to Cloudflare Pages as a static site. Push to main to trigger a deployment.
